Passos para a criação do site
- criar index.html
digitar ! e enter para preenchimento automático
mudar o título do <head>
- criar style.css
- usar a tag <link> dentro da seção <head> no html para acessar o arquivo css
<link rel="stylesheet" href="style.css">
- acessar https://getbootstrap.com/
escolher entre fazer o download dos arquivos CSS e JavaScript do Bootstrap no computador e adicioná-los ao 
projeto (Install via package manager) ou adicionar um link para o Bootstrap hospedado na nuvem em um arquivo 
HTML (Include via CDN). essa segunda opção é a mais aconselhada.
copiar o link css e colar na seção <head>
copiar o link js e colar na seção <head> (se o site tiver interação com o usuário)
- no site do bootstrap, procurar pelos components que serão usados no projeto
